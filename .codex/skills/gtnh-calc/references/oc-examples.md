# 超频计算示例

## 核心规则

OC 档数 = 供电电压 index - 配方原始电压 index。同等级 = 0 档，无 OC。

## 有损 OC（单方块机器）

每档：功率 x4，时间 /2。可叠多档。

### 示例 1：LV 配方 x HV 供电

- 配方：2 EU/t, 15s @ LV (index=1)
- 供电：HV (index=3)
- 档数：3-1 = 2 档
- 结果：EU = 2 x 16 = 32 EU/t, 时间 = 15/4 = 3.75s

### 示例 2：MV 配方 x HV 供电

- 配方：128 EU/t, 22.4s @ MV (index=2)
- 供电：HV (index=3)
- 档数：3-2 = 1 档
- 结果：EU = 128 x 4 = 512 EU/t, 时间 = 22.4/2 = 11.2s

## 无损 OC（LCR 等部分多方块）

每档：功率 x4，时间 /4。可叠多档。

### 示例 3：LV 配方 x HV LCR

- 配方：32 EU/t, 24s @ LV (index=1)
- 供电：HV LCR (index=3), 无损 OC
- 档数：3-1 = 2 档
- 结果：EU = 32 x 16 = 512 EU/t, 时间 = 24/16 = 1.5s

### 示例 4：LV 配方 x MV LCR

- 配方：128 EU/t, 8s @ LV (index=1)
- 供电：MV LCR (index=2), 无损 OC
- 档数：2-1 = 1 档
- 结果：EU = 128 x 4 = 512? 等，看实际 base
- 结果：time = 8/4 = 2s

## 常见错误

1. **认为有损 OC 只能 1 档**：可叠多档。LV x HV = 2 档。
2. **混淆无损/有损除数**：无损除 4，有损除 2。
3. **同等级也 OC**：配方 LV + LV 供电，0 档，无变化。
4. **功率公式去触 OC**：正确是电压差定档数，不是功率除以。
5. **把无损当所有多方块**：仅部分多方块支持无损 OC，逐一确认。
